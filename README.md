## 🚀 Featured work

### 🧠 NEURA — Brain Tumor Segmentation
A deep-learning system for 3D MRI tumor segmentation. I trained an **nnU-Net v2** model on 350 BraTS 2021 cases across four MRI modalities, reaching **0.92 Whole-Tumor Dice**, and shipped it as a web app that returns color-coded segmentation overlays alongside the evaluation metrics.
`PyTorch` `nnU-Net v2` `Flask`
<sub>🔒 Repo opens after degree completion (academic policy)</sub>

### 📊 Operational Analytics Platform
A single-window platform that replaced a pile of fragmented manual reporting. A **FastAPI + DuckDB-over-Parquet** backend serves sub-500ms reads, fed by a three-stage pipeline, with an optimized extraction query over **~16M rows** verified to produce byte-identical output to the original.
`FastAPI` `DuckDB` `Parquet`
<sub>💼 Built in a professional setting</sub>
